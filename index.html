Basic tags

Tag Name Description
<head> head element Sets off the title and information to be included that is not for viewers
<body> body element Contains all the content for viewers
<title> title element Puts name of the document in the title bar
<p> p element Defines a new paragraph
<h1> - <h6> headings Defines headlines; h1 = largest, h6 = smallest
<br> line break Start a new line
<div> div element Use for format block content
<span> span element Use for format inline content

Formatting

Tag Name Description
<strong> strong element Emphasizes a word, processed in bold
<em> em element Emphasizes a word, processed in italic

Lists

Tag Name Description
<ul> unordered list Creates an unordered list, bullets point
<ol> ordered list Creates an ordered list, marked with numbers
<li> list Encompasses each list item

Tables

Tag Name Description
<table> table element Defines a table
<th> table header Creates table header and text in this elements are bold and centered
<tbody> table body Defines the body of a table
<tr> table row Defines each row in the table
<td> table data Defines the content of the table cell
